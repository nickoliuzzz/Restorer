## Результаты тестирования

#### Возможность авторизации

1. 001

2. Проверка возможности регистрации;

3. Произведите следующие шаги:

   1. Перейти на страницу регистрация.

   2. Введите почту.

   3. Введите пароль, повторный пароль.

   4. Нажмите кнопку "Зарегистрироваться".

4. Переход на страницу "Подтвердите e-mail". При некоректном вводе данных появится окно с сообщением о ошибке.

##### Результат: соответствует требованиям.

#### Возможность авторизации

1. 002

2. Проверка возможности авторизации;

3. Произведите следующие шаги:

   1. Перейти на страницу авторизации.

   2. Введите логин.

   3. Введите пароль.

   4. Нажмите кнопку "Вход".

4. Переход на главную страницу сайта. При неуспешной авторизации появится окно с сообщением "Неверный логин или пароль"

##### Результат: соответствует требованиям.

#### Возможность финансировать проекты

1. 003

2. Проверка возможности финансирование действующих проектов;

3. Произведите следующие шаги:

   1. Авторизированным перейти на страницу любого проекта.

   2. Нажмите кнопку "Профинансировать".

   3. Введите нужную сумму в диалоговое окно.

4. Переход на главную страницу сайта.

##### Результат: соответствует требованиям.

#### Возможность подать заявку на повышение привелегий

1. 004

2. Проверка возможности на повышение привелегий.

3. Произведите следующие шаги:

   1. Авторизированным перейти в профиль.

   2. Загрузки документ.
 
4. В профиле появится надпись "Ожидание решение администратора".

##### Результат: соответствует требованиям.

#### Возможность создать проект

1. 005

2. Создание проекта пользователем.

3. Произведите следующие шаги:

   1. Привилегированным пользователем перейти в профиль.

   2. Нажать "Создать проект".

   3. Заполнить поля, вставить картинку, нажать "Создать".

   4. Заполнить контент, нажать "Создать".

   5. Выбрать цели, нажать "Создать".
 
4. Переход на главную страницу.

##### Результат: соответствует требованиям.

#### Возможность редактировать проект

1. 006

2. Проверка редактирование проекта.

3. Произведите следующие шаги:

   1. Авторизированным зайти в свой проект.

   2. Нажать "Редактировать".

   3. Заполнить данные.

   4. Нажать "Создать".

4. Переход на главную страницу.

##### Результат: соответствует требованиям.

#### Возможность получить уведомление о окончание компании сбора средств на проект

1. 007

2. Удостовериться в получение уведомления о окончание сбора средст

3. Произведите следующие шаги:

   1. Создать проект.

   2. Дождаться окончания проекта.

   3. Проверить почту.

4. Письмо с ссылкой на список пользователей, которые профнансировали проект.

##### Результат: соответствует требованиям.

#### Возможность просмотра всех проектов на страницу "Все проекты"

1. 008

2. Удостоверится в просмотре всех проектов.

3. Произведите следующие шаги:

   1. Перейти на страницу "Все проекты".

   2. Проскроллить вниз.
   
4. Должны появляться(подгружаться) новые проекты.

##### Результат: соответствует требованиям.

#### Возможность для администратора проверки заявок

1. 009

2. Удостоверится в работе одобрения/отклонения заявок пользователей.

3. Произведите следующие шаги:

   1. Авторизоваться в качестве администратора.

   2. Перейти на страницу "Пользователи".
   
   3. Нажать на кнопку "Await".
   
   4. Принять/отклонить заявку.
   
4. После перезагрузки страницы пользователь будет помечен как прелигированный(True), либо обратное(False).

##### Результат: соответствует требованиям.

#### Возможность для администратора удалять/блокировать/разблокировать пользователей

1. 010

2. Удостоверится в работе удаления/блокировки/разблокировки пользователей.

3. Произведите следующие шаги:

   1. Авторизоваться в качестве администратора.

   2. Перейти на страницу "Пользователи".
   
   3. Выбрать пользователей с помощью checkbox.
   
   4. Нажать на кнопку "Удалить"/"Заблокировать"/"Разблокировать/.
   
4. После перезагрузки страницы пользователь который удален - пропадет из списка, заблокированный - выделится желтым, разблокированный - прозрачный.

##### Результат: соответствует требованиям.

#### Возможность для администратора удалять/редактировать проекты

1. 011

2. Удостоверится в работе редактирования проектов.

3. Произведите следующие шаги:

   1. Авторизоваться в качестве администратора.

   2. Перейти на любой проект.
   
   3. Нажать "Редактировать".

   4. Заполнить данные.

   5. Нажать "Создать".
 
4. Переход на главную страницу.

##### Результат: соответствует требованиям.

#### Возможность смены языка

1. 012

2. Удостоверится в работе смены локализовки.

3. Произведите следующие шаги:

   1. Выбрать язык на любой страницу проекта.

4. Смена языка на текущей странице

##### Результат: не соответствует требованиям. Выдает ошибку, если сменить язык на странице проекта.

#### Возможность смены шаблона сайта

1. 013

2. Удостоверится в работе смены темы сайта на темную/светлую.

3. Произведите следующие шаги:

   1. Нажать на кнопку изменения стилизации на любой странице сайта.

4. Переход на главную страницу с иной цветовой схемой.

##### Результат: соответствует требованиям.
